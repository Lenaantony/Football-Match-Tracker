# Football Match Tracker ⚽

A Python-based football tracker that allows users to:

- View teams
- Search team information
- Save favourite teams
- View fixtures and standings (coming soon)

## Features

- Team search
- Team information
- Menu-driven interface
- File handling
- API integration (planned)

## Technologies

- Python
- Git
- GitHub

## Future Improvements

- Live football API integration
- Match fixtures
- League standings
